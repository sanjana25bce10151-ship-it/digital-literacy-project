# Project Overview
This repository serves as a Digital Literacy Portfolio created as part of my role as a Student Digital Ambassador. The objective of this project is to demonstrate foundational digital skills, including professional online presence, secure communication, and cybercrime awareness, to help peers navigate the digital world safely and effectively.

# Module Summaries
This project is organized into five key modules, each documented within this repository:

# Task 1: Digital Literacy Awareness Infographic
An educational visual resource created using Canva to explain the importance of digital literacy and safe internet practices.

# Task 2: Student Digital Portfolio
Professional profiles established on GitHub, LinkedIn, and Stack Overflow to build a visible academic and career identity.

# Task 3: Coding & Collaboration
Demonstration of technical practice via HackerRank and a custom-built Digital Literacy Awareness Quiz for peer education.

# Task 4: Email Etiquette & Social Media
Formal communication drafts and a comprehensive "Do's and Don'ts" guide for responsible social media use.

# Task 5: Cybercrime Awareness
A detailed case study on UPI/Online Payment Fraud and a step-by-step prevention guide for students.
